IG Web API .NET Sample Trading Applications

Overview

The IG Web API .NET Sample Trading Applications package contains a portable class library (SDK) and two reference applications. Full source is available for the portable class library as well as the two reference applications.
  1. The portable class library (PublicPcl) contains a Rest API for static data and an asynchronous streaming API for dynamic data.
  2. The reference applications (Windows Forms and WPF) demonstrate how to:
    • log into your demo or live account via the Rest API
    • retrieve positions, watchlists and market orders via the Rest API
    • search, and browse markets via the Rest API
    • stream live data via the Lightstreamer API.

Installation

  1. Using Visual Studio 2012 or Visual Studio 2013, open the solution file IGPublicSDK.sln containing the three projects: IGPublicPcl, SampleWpfTrader and SampleWinFormTrader.
  2. Add references to the LightStreamer (DotNetClient_N2) and Json.net (Newtonsoft.Json) libraries:
    • the LightStreamer asynchronous Streaming API(DotNetClient_N2) is contained in the 3rdPartyDlls directory
    • Json.net (Newtonsoft.Json) is installable using NuGet
  3. Build the solution, then set either the SampleWinFormTrader or SampleWPFTrader as the start project.
  4. Check which IG environment you want to access, and set this to the correct value. The environment is set in the PublicPcl project in the IGRestService.cs file in the IgRestService class: parameter BaseUrl.
  5. Before you run your project, you will need to set the API key, your user identifier and password.
    • In the SampleWPFTrader application, you set these in the ApplicationViewModel.cs in the Login method.
    • In the SampleWinFormTrader application, you set these in the IGApiClientApplication.cs file, in the DemoDotNetClientForm constructor.

Upgrading

  1. Should you choose to upgrade to a newer version of the IG Web API .NET Trading Applications package, we recommend importing the new portable class library (PublicPcl) to your existing project making changes as required for this new API.
